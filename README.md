# Java Intermediate Mini Systems

This repository contains intermediate Java practice projects focused on strengthening object-oriented programming, collections, file handling, exceptions, and threading concepts.

## Current Project: Task Manager With Priorities

The first project is a console-based task manager that allows users to create, manage, filter, and save tasks.

## Concepts Practiced

- Packages
- Classes and objects
- Encapsulation
- Getters and setters
- Constructors
- Enums
- LinkedList
- Iterators
- File handling
- Service classes
- Utility classes
- Basic project structure
